How do we get Pokemon Data?

Various Pokemon-related information is retrieved by the API website pokeapi.co

The data is retrieved Pokemon-by-Pokemon using Python crawler under the API format e.g. pokeapi/ditto/

The data are formatted in json file, which they can be serialized by django for data insertion.

How do we get the sprites and pictures?

There are two types of Pokemon sprites used in this project: mini ones and bigger ones.
Sprites are called by icon tags after their link references are included in the project.

mini sprites by Lunik: https://github.com/Lunik/pokemon-icon

bigger sprites by karismatic-megafauna: https://github.com/karismatic-megafauna/pokemon-images
